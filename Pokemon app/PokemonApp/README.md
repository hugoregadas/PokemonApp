# Swift-MVVM-Pokemons

Está aplicação recebe e trata informação de uma api (https://pokeapi.co/) que retorna dados de pokemon. 

 ## Desenvolvimento

1. Aplicação desenvolvida em Swift 
2. Arquitectura MVVM
3. Testes Unitários
4. DarkMode
5. Permite Mudar a linguagem da app
